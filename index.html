No ads. No tracking. Open source.

Twt has no ads and will never track its users. On Twt you own your content and you are free to run your own pod on a server you control.

Decentralized internet

Your social media the way it should be.

Open source project Twt is a decentralized micro-blogging platform that adheres to a simple standard called twtxt. You can download it and run you own.

Hosted instances Not a technical person? No worries. The creators of this project also created Twt.social where you can start your own small community and join the larger Twt “social media” network with family, friends and like minded folks. We bring the tech, you bring the conversations. Learn more at Twt.social.

RESTful API Use our API to build your own mobile api or CLI command application that integrates with txtxt.

How does Twt compare to Twitter™?

Twt and thus all Twt.social pods (that’s what we call an individual server) is actually pretty similar to Twitter™ so if you’ve used Twitter™ before you’ll feel right at home!
